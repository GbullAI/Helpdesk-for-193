1. First of all, check your internet connection of the computer you are currently using.
  • Go to Start
  • Search Control Panel
  • Click on Network and Sharing Center
  • Verify that one of the active network has Access Types: Internet
   
    2. If there is no internet access , please check with your IT department of your branch.
     
  • Go to Start
  • Type in cmd to open Command Prompt
  • Type in ping nvis.jpph.gov.my
  • The ping result should return 10.23.171.200
If all of those steps passed without any issue, please escalate the problem to our helpdesk.
